# Live Coding – Automation Testing (Java 21 + Gradle)

This repository is a **template** for automation live coding interviews.

Supported tasks:
- API test automation
- UI automation (pseudo or real)
- Test framework design
- Test design (ISTQB)

Use the following files depending on the task:

- API → `src/test/java/interview/api/ApiTest.java`
- UI → `src/test/java/interview/ui/UiTestExample.java`
- Framework → `src/test/java/interview/framework/FrameworkSketch.java`
- Test Design → `src/test/java/interview/design/TestDesign.md`
